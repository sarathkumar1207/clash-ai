Architecture

Clean boundaries for a legal companion

Presentation

React, Router, TanStack Query, Zustand, Tailwind, skeleton states, and responsive dashboard views.

Application

Upgrade ranking, clan aggregation, AI prompt orchestration, screenshot analysis jobs, and progress workflows.

Infrastructure

Official Clash API client, Turso repositories, GitHub OAuth, OpenRouter/Ollama adapters, OpenCV/ONNX vision adapters.

Deployment

GitHub Pages for the frontend, free-tier Rust backend options, and CI checks for Rust and frontend builds.
