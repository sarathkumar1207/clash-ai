# Environment Variables

| Name | Required | Description |
| --- | --- | --- |
| `CLASH_API_TOKEN` | yes for live API | Official Clash of Clans API token |
| `CLASH_API_BASE_URL` | no | Defaults to `https://api.clashofclans.com/v1` |
| `TURSO_DATABASE_URL` | yes for persistence | Turso SQLite URL |
| `TURSO_AUTH_TOKEN` | yes for persistence | Turso auth token |
| `GITHUB_CLIENT_ID` | yes for OAuth | GitHub OAuth app client id |
| `GITHUB_CLIENT_SECRET` | yes for OAuth | GitHub OAuth secret |
| `OPENROUTER_API_KEY` | optional | Hosted AI provider key |
| `OLLAMA_BASE_URL` | optional | Local Ollama endpoint |
| `VISION_MODEL_PATH` | optional | ONNX model path |
